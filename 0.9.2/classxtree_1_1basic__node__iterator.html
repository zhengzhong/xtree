Detailed Description

template<class T>
class xtree::basic_node_iterator< T >

This class template represents an iterator to an XML node object. Increasing the iterator will move to the next sibling node of the current node; decreasing the iterator will move to the previous sibling node of the current node. It is required the template parameter T provide the following member functions:

 // Required by basic_node_iterator<T>:
 basic_node_ptr<T> T::next_sibling();
 basic_node_ptr<T> T::prev_sibling();

 // Required by basic_node_iterator<const T>:
 basic_node_ptr<const T> T::next_sibling() const;
 basic_node_ptr<const T> T::prev_sibling() const;

This class template is used by child_node_list and attribute_map.

This class template is implemented in term of basic_node_ptr class template.
