Detailed Description

This class represents a child node list under an element or a document. It may be viewed as a container of pointers to child_node. This class provides a similar API as std::list.

iterator xtree::child_node_list::erase ( iterator  first,
iterator  last 
)

Removes and destroys all the child nodes between the first position (included) and the last position (excluded). This function has undefined behavior if the first iterator cannot reach the last iterator by increasing itself.

Postcondition:
size() is N less (N is the number of child nodes erased).
Parameters:
firstthe first position (included) of the child node to erase.
lastthe last position (excluded) of the child node to erase.
Returns:
an iterator to the child node after the last erased one (thus, ==last).
