Detailed Description

This class represents an XML DOM parser. A parser is used to parse an XML file or string and return the document object.

document* xtree::dom_parser::parse_file ( const std::string &  file_name)

Parses an XML file to a document object. It is up to the user to free the document object. It is recommended that user use the RAII idiom and put the returned document into a smart pointer (std::auto_ptr or boost::shared_ptr).

Parameters:
file_namethe name of the xml file to parse.
Returns:
a document object parsed from the file, never null.
Exceptions:
dom_errorif fail to parse the XML file.
