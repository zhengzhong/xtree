00111         //! Declares a namespace on this element. The namespace URI is provided as an argument,
00112         //! while the namespace prefix is generated automatically.
00113         //!
00114         //! Note: in an XML document, namespace prefix is only an alias to a certain namespace URI,
00115         //! and different prefixes mapped to the same URI are semantically identical. Thus, to
00116         //! avoid prefix conflicts and ambiguity, each time a namespace URI is declared on the
00117         //! element, a new prefix will be generated automatically, and it is guaranteed to be
00118         //! unique in the document scope. Users are not allowed to choose a customized namespace
00119         //! prefix.
00120         //!
00121         //! \param uri  the namespace URI to declare.
00122         //! \return the auto-generated prefix mapped to the namespace URI.
00123         //! \throw bad_dom_operation  if the namespace URI is invalid.
